## このリポジトリについて

中学受験の問題集・参考書は種類が膨大で、「結局うちの子にはどれがいいの？」と迷う保護者がほとんどです。

このデータベースは、各科目を**「基礎 → 標準 → 応用 → 過去問演習」**の一本道で整理し、**取り組む順番・使い方・学習ペースの目安**まで含めてまとめています。塾なし（自宅学習中心）で中学受験に挑む家庭にも、2科目受験を考えているご家庭にも、塾の補助教材を探している家庭にも活用いただけます。

## 学習の全体像

```
【基礎フェーズ】各科目の土台を作る
    ↓
【標準フェーズ】中学受験レベルの学力を身につける
    ↓
【応用フェーズ】弱点を潰し、得点力を高める
    ↓
【過去問フェーズ】志望校対策（小6夏以降）
```

目標は**小6の夏休み前**までに標準フェーズを完了すること。その後は志望校の過去問を中心に、弱点補強と実戦演習に切り替えます。

## 学習の5ステップ（全科目共通）

問題集を使った勉強の基本手順です。どの科目・どの教材でもこの流れで進めます。

1. **ノートに解く** — 問題集に直接書き込まない（繰り返し使うため）
2. **丸つけをする** — 「できた」と「できなかった」を印で明確に分ける
3. **間違えた問題を理解する** — 解説をよく読み、なぜ間違えたかを把握する
4. **その場でもう一度解く** — 解答を伏せて、自力で解けるか確認する
5. **後日、復習する** — 印のついた問題を再度解いて定着を確認する

> この「できない → わかる → できる」のサイクルが学力向上の核心です。
