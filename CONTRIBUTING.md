# コントリビューションガイド

このリポジトリへの情報提供を歓迎します！ 以下のルールに沿ってご協力ください。

## 追加できる情報

- **問題集・参考書の推薦：** 実際に使用した経験に基づくもの
- **既存情報の修正：** 改訂版の発売、絶版情報、誤記の修正など
- **使い方のTips：** 特定の教材の効果的な使い方

## 追加のフォーマット

問題集を追加する場合は、以下のフォーマットに従ってください。

```markdown
| 書名 | 出版社 | 偏差値帯 | 一言コメント |
```

偏差値帯は以下の絵文字で表記します。

- 🟢 基礎（〜45）
- 🔵 標準（45〜55）
- 🟡 応用（55〜65）
- 🔴 発展（65〜）

## 手順

1. このリポジトリを Fork する
2. 新しいブランチを作成する（`git checkout -b add-book-xxxxx`）
3. 該当する科目のファイルに情報を追加する
4. Pull Request を送る

**Issue だけでもOK：** Git操作に慣れていない方は、[Issue](../../issues) で「この本を追加してほしい」と書いていただくだけでも大丈夫です。

## お願い

- アフィリエイトリンクは含めないでください
- 実際に使用・指導した経験に基づく推薦をお願いします
- 著作権に配慮し、書籍の内容そのものは掲載しないでください
